# CodeBlock

Syntax-highlighted code with copy button.

```tsx
import { CodeBlock } from '@ybouhjira/hyperkit';
```

## Playground

Edit the code — the preview recompiles and re-renders as you type.

## More Examples

### Bash

```tsx
<CodeBlock
  code={"#!/bin/bash\n\n# Install dependencies\nnpm install\n\n# Run tests\nnpm test\n\n# Build for production\nnpm run build"}
  language="bash"
  label="build.sh"
  showLineNumbers
/>
```

### HTML

```tsx
<CodeBlock
  code={"<!DOCTYPE html>\n<html lang=\"en\">\n  <head>\n    <meta charset=\"UTF-8\">\n    <meta name=\"viewport\" content=\"width=device-width, initial-scale=1.0\">\n    <title>SolidKit Demo</title>\n  </head>\n  <body>\n    <div id=\"root\"></div>\n    <script type=\"module\" src=\"/src/index.tsx\"></script>\n  </body>\n</html>"}
  language="html"
  label="index.html"
/>
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `code` * | `string` | — | Code content to display. |
| `language` | `string` | — | Syntax highlighting language. If omitted, auto-detects language. |
| `label` | `string` | — | Optional label shown in the header. |
| `showLineNumbers` | `boolean` | `false` | Show line numbers in the gutter. |
| `class` | `string` | — | Additional CSS classes. |
| `style` | `JSX.CSSProperties` | — | Custom styles. |

`*` required prop.
