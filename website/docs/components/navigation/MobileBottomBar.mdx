## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `items` * | `MobileBottomBarItem[]` | — | Navigation items rendered left-to-right. 2-5 items is the recommended range. |
| `activeId` | `string` | — | Currently active item id. If unset, no item is highlighted. |
| `onSelect` | `(id: string) => void` | — | Called when a user taps a non-disabled item. |
| `hidden` | `boolean` | `false` | When true, the bar renders nothing. The primitive itself is viewport-agnostic — consumers use `useBreakpoint()` (or their own logic) to decide when to hide it on larger screens. |
| `'aria-label'` | `string` | `'Primary'` | Accessible label for the \<nav> landmark. |
| `class` | `string` | — | Additional class applied to the root \<nav>. |
| `style` | `JSX.CSSProperties` | — | Inline style merged onto the root \<nav>. |

`*` required prop.
