# AspectRatio

Maintains the aspect ratio of its child.

```tsx
import { AspectRatio } from '@ybouhjira/hyperkit';
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `ratio` | `number` | `1` | Width-to-height ratio (e.g., 16/9, 4/3, 1). |
