# Layout

Structural primitives for arranging content.

| Component                          | Description                                                            |
| ---------------------------------- | ---------------------------------------------------------------------- |
| [AspectRatio](./AspectRatio.mdx)   | Maintains the aspect ratio of its child.                               |
| [Box](./Box.mdx)                   | Base polymorphic container with spacing and color tokens.              |
| [Center](./Center.mdx)             | Centers content on both axes.                                          |
| [Container](./Container.mdx)       | Max-width page wrapper with padding.                                   |
| [DocumentPage](./DocumentPage.mdx) | A4/letter page container with margin simulation.                       |
| [Flex](./Flex.mdx)                 | Flexbox container with direction, align, justify, gap, and wrap props. |
| [Grid](./Grid.mdx)                 | CSS grid with columns, gap, and auto-fit/fill.                         |
| [MasonryGrid](./MasonryGrid.mdx)   | Pinterest-style masonry layout.                                        |
| [MediaGrid](./MediaGrid.mdx)       | Responsive media card grid.                                            |
| [ScrollArea](./ScrollArea.mdx)     | Custom-scrollbar scroll container.                                     |
| [Section](./Section.mdx)           | Labeled page section with a heading.                                   |
| [Spacer](./Spacer.mdx)             | Flexible space filler for flex and grid layouts.                       |
| [Stack](./Stack.mdx)               | Vertical flex column with a consistent spacing token.                  |
| [Wrap](./Wrap.mdx)                 | Flex-wrap container for tag and chip layouts.                          |
