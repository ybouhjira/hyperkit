# Layout

Structural primitives for arranging content.

| Component | Description |
| --------- | ----------- |
| [AspectRatio](./AspectRatio.md) | Maintains the aspect ratio of its child. |
| [Box](./Box.md) | Base polymorphic container with spacing and color tokens. |
| [Center](./Center.md) | Centers content on both axes. |
| [Container](./Container.md) | Max-width page wrapper with padding. |
| [DocumentPage](./DocumentPage.md) | A4/letter page container with margin simulation. |
| [Flex](./Flex.md) | Flexbox container with direction, align, justify, gap, and wrap props. |
| [Grid](./Grid.md) | CSS grid with columns, gap, and auto-fit/fill. |
| [MasonryGrid](./MasonryGrid.md) | Pinterest-style masonry layout. |
| [MediaGrid](./MediaGrid.md) | Responsive media card grid. |
| [ScrollArea](./ScrollArea.md) | Custom-scrollbar scroll container. |
| [Section](./Section.md) | Labeled page section with a heading. |
| [Spacer](./Spacer.md) | Flexible space filler for flex and grid layouts. |
| [Stack](./Stack.md) | Vertical flex column with a consistent spacing token. |
| [Wrap](./Wrap.md) | Flex-wrap container for tag and chip layouts. |
