## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `size` | `SpaceToken \| string \| number` | — | Fixed size. If omitted, uses flex:1 to fill available space. |
| `axis` | `'horizontal' \| 'vertical'` | `'horizontal'` | Which dimension to space. |
