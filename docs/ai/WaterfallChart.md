# WaterfallChart

> Component

## Props

| Prop         | Type                                   | Required | Default | Description |
| ------------ | -------------------------------------- | -------- | ------- | ----------- |
| `items`      | `WaterfallItem[]`                      | Yes      | -       | -           |
| `height`     | `number`                               | No       | -       | -           |
| `labelWidth` | `number`                               | No       | -       | -           |
| `tickCount`  | `number`                               | No       | -       | -           |
| `class`      | `string`                               | No       | -       | -           |
| `style`      | `import('solid-js').JSX.CSSProperties` | No       | -       | -           |
