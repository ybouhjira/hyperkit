# Accordion

> Component

## Props

| Prop           | Type                  | Required    | Default | Description |
| -------------- | --------------------- | ----------- | ------- | ----------- | --- |
| `items`        | `AccordionItemData[]` | Yes         | -       | -           |
| `type`         | `'single'             | 'multiple'` | No      | -           | -   |
| `defaultValue` | `string               | string[]`   | No      | -           | -   |
| `collapsible`  | `boolean`             | No          | -       | -           |
| `disabled`     | `boolean`             | No          | -       | -           |
| `class`        | `string`              | No          | -       | -           |
| `style`        | `JSX.CSSProperties`   | No          | -       | -           |
| `unstyled`     | `boolean`             | No          | -       | -           |

## Examples

### Basic Usage

```tsx
import { Accordion } from '@ybouhjira/hyperkit';

<Accordion>Content</Accordion>;
```
