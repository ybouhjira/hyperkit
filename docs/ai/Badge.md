# Badge

> Component

## Props

| Prop       | Type                | Required  | Default   | Description |
| ---------- | ------------------- | --------- | --------- | ----------- | ------- | --- | --- | --- |
| `variant`  | `'default'          | 'success' | 'warning' | 'danger'    | 'info'` | No  | -   | -   |
| `type`     | `'label'            | 'dot'     | 'count'`  | No          | -       | -   |
| `count`    | `number`            | No        | -         | -           |
| `maxCount` | `number`            | No        | -         | -           |
| `children` | `JSX.Element`       | No        | -         | -           |
| `class`    | `string`            | No        | -         | -           |
| `style`    | `JSX.CSSProperties` | No        | -         | -           |
| `unstyled` | `boolean`           | No        | -         | -           |

## Examples

### Basic Usage

```tsx
import { Badge } from '@ybouhjira/hyperkit';

<Badge>Content</Badge>;
```
