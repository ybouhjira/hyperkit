# Timeline

> Component

## Props

| Prop          | Type                | Required      | Default | Description |
| ------------- | ------------------- | ------------- | ------- | ----------- | --- | --- |
| `steps`       | `TimelineStep[]`    | Yes           | -       | -           |
| `orientation` | `'vertical'         | 'horizontal'` | No      | -           | -   |
| `size`        | `'sm'               | 'md'          | 'lg'`   | No          | -   | -   |
| `class`       | `string`            | No            | -       | -           |
| `style`       | `JSX.CSSProperties` | No            | -       | -           |
