# Skeleton

> Component

## Props

| Prop       | Type      | Required | Default | Description |
| ---------- | --------- | -------- | ------- | ----------- | --- | --- |
| `variant`  | `'rect'   | 'circle' | 'text'` | No          | -   | -   |
| `width`    | `string   | number`  | No      | -           | -   |
| `height`   | `string   | number`  | No      | -           | -   |
| `size`     | `number`  | No       | -       | -           |
| `lines`    | `number`  | No       | -       | -           |
| `class`    | `string`  | No       | -       | -           |
| `unstyled` | `boolean` | No       | -       | -           |

## Examples

### Basic Usage

```tsx
import { Skeleton } from '@ybouhjira/hyperkit';

<Skeleton>Content</Skeleton>;
```
