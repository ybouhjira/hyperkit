# Switch

> Component

## Props

| Prop             | Type                         | Required | Default | Description |
| ---------------- | ---------------------------- | -------- | ------- | ----------- | --- |
| `checked`        | `boolean`                    | No       | -       | -           |
| `defaultChecked` | `boolean`                    | No       | -       | -           |
| `onChange`       | `(checked: boolean) => void` | No       | -       | -           |
| `disabled`       | `boolean`                    | No       | -       | -           |
| `label`          | `string`                     | No       | -       | -           |
| `description`    | `string`                     | No       | -       | -           |
| `size`           | `'sm'                        | 'md'`    | No      | -           | -   |
| `class`          | `string`                     | No       | -       | -           |
| `unstyled`       | `boolean`                    | No       | -       | -           |

## Examples

import { Switch } from '@ybouhjira/hyperkit';
