# Sparkline

> Component

## Props

| Prop          | Type                                   | Required | Default | Description |
| ------------- | -------------------------------------- | -------- | ------- | ----------- |
| `data`        | `number[]`                             | Yes      | -       | -           |
| `width`       | `number`                               | No       | -       | -           |
| `height`      | `number`                               | No       | -       | -           |
| `color`       | `string`                               | No       | -       | -           |
| `strokeWidth` | `number`                               | No       | -       | -           |
| `filled`      | `boolean`                              | No       | -       | -           |
| `animate`     | `boolean`                              | No       | -       | -           |
| `class`       | `string`                               | No       | -       | -           |
| `style`       | `import('solid-js').JSX.CSSProperties` | No       | -       | -           |

## Examples

### Basic Usage

```tsx
import { Sparkline } from '@ybouhjira/hyperkit';

<Sparkline>Content</Sparkline>;
```
