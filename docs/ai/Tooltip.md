> Component

| Prop        | Type          | Required     | Default | Description |

| `content`   | `string       | JSX.Element` | Yes     | -           | -   |
| `placement` | `'top'        | 'bottom'     | 'left'  | 'right'`    | No  | -   | -   |
| `delay`     | `number`      | No           | -       | -           |
| `children`  | `JSX.Element` | Yes          | -       | -           |
| `class`     | `string`      | No           | -       | -           |
| `unstyled`  | `boolean`     | No           | -       | -           |
