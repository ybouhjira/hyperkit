# StatBar

> Component

## Props

| Prop        | Type                | Required    | Default | Description |
| ----------- | ------------------- | ----------- | ------- | ----------- | --- | --- |
| `items`     | `StatBarItem[]`     | Yes         | -       | -           |
| `size`      | `'sm'               | 'md'        | 'lg'`   | No          | -   | -   |
| `direction` | `'horizontal'       | 'vertical'` | No      | -           | -   |
| `class`     | `string`            | No          | -       | -           |
| `style`     | `JSX.CSSProperties` | No          | -       | -           |
| `unstyled`  | `boolean`           | No          | -       | -           |

## Examples

### Basic Usage

```tsx
import { StatBar } from '@ybouhjira/hyperkit';

<StatBar>Content</StatBar>;
```
