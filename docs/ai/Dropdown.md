# Dropdown

> Component

## Props

| Prop       | Type             | Required | Default | Description |
| ---------- | ---------------- | -------- | ------- | ----------- |
| `items`    | `DropdownItem[]` | Yes      | -       | -           |
| `trigger`  | `JSX.Element`    | Yes      | -       | -           |
| `class`    | `string`         | No       | -       | -           |
| `unstyled` | `boolean`        | No       | -       | -           |

## Examples

### Basic Usage

```tsx
import { Dropdown } from '@ybouhjira/hyperkit';

<Dropdown>Content</Dropdown>;
```
