# Grid

> Component

## Props

| Prop         | Type         | Required | Default  | Description |
| ------------ | ------------ | -------- | -------- | ----------- | --- | --- | --- |
| `columns`    | `number      | string`  | No       | -           | -   |
| `rows`       | `number      | string`  | No       | -           | -   |
| `gap`        | `SpaceToken` | No       | -        | -           |
| `rowGap`     | `SpaceToken` | No       | -        | -           |
| `columnGap`  | `SpaceToken` | No       | -        | -           |
| `autoFlow`   | `'row'       | 'column' | 'dense'` | No          | -   | -   |
| `placeItems` | `'start'     | 'center' | 'end'    | 'stretch'`  | No  | -   | -   |
| `areas`      | `string`     | No       | -        | -           |

## Examples

### Basic Usage

```tsx
import { Grid } from '@ybouhjira/hyperkit';

<Grid>Content</Grid>;
```
