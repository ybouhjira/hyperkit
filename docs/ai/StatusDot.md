# StatusDot

> Component

## Props

| Prop         | Type                | Required  | Default   | Description |
| ------------ | ------------------- | --------- | --------- | ----------- | ------- | --- | --- | --- |
| `status`     | `'default'          | 'success' | 'warning' | 'danger'    | 'info'` | No  | -   | -   |
| `size`       | `'sm'               | 'md'      | 'lg'`     | No          | -       | -   |
| `pulse`      | `boolean`           | No        | -         | -           |
| `label`      | `string`            | No        | -         | -           |
| `aria-label` | `string`            | No        | -         | -           |
| `class`      | `string`            | No        | -         | -           |
| `style`      | `JSX.CSSProperties` | No        | -         | -           |
| `unstyled`   | `boolean`           | No        | -         | -           |

## Examples

### Basic Usage

```tsx
import { StatusDot } from '@ybouhjira/hyperkit';

<StatusDot>Content</StatusDot>;
```
