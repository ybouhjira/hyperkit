# Tabs

> Component

## Props

| Prop          | Type                      | Required    | Default | Description |
| ------------- | ------------------------- | ----------- | ------- | ----------- | --- |
| `items`       | `TabItem[]`               | Yes         | -       | -           |
| `value`       | `string`                  | No          | -       | -           |
| `onChange`    | `(value: string) => void` | No          | -       | -           |
| `orientation` | `'horizontal'             | 'vertical'` | No      | -           | -   |
| `class`       | `string`                  | No          | -       | -           |

## Examples

### Basic Usage

```tsx
import { Tabs } from '@ybouhjira/hyperkit';

<Tabs>Content</Tabs>;
```
