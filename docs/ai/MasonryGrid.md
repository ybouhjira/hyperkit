# MasonryGrid

> Component

## Props

| Prop       | Type                | Required                                                | Default | Description |
| ---------- | ------------------- | ------------------------------------------------------- | ------- | ----------- | --- |
| `columns`  | `number             | { sm?: number; md?: number; lg?: number; xl?: number }` | No      | -           | -   |
| `gap`      | `SpaceToken`        | No                                                      | -       | -           |
| `class`    | `string`            | No                                                      | -       | -           |
| `style`    | `JSX.CSSProperties` | No                                                      | -       | -           |
| `children` | `JSX.Element`       | No                                                      | -       | -           |

## Examples

### Basic Usage

```tsx
import { MasonryGrid } from '@ybouhjira/hyperkit';

<MasonryGrid>Content</MasonryGrid>;
```
