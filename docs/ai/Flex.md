# Flex

> Component

## Props

| Prop        | Type         | Required | Default         | Description       |
| ----------- | ------------ | -------- | --------------- | ----------------- | ----------- | --------- | --- | --- | --- |
| `direction` | `'row'       | 'column' | 'row-reverse'   | 'column-reverse'` | No          | -         | -   |
| `align`     | `'start'     | 'center' | 'end'           | 'stretch'         | 'baseline'` | No        | -   | -   |
| `justify`   | `'start'     | 'center' | 'end'           | 'between'         | 'around'    | 'evenly'` | No  | -   | -   |
| `gap`       | `SpaceToken` | No       | -               | -                 |
| `wrap`      | `'nowrap'    | 'wrap'   | 'wrap-reverse'` | No                | -           | -         |
| `inline`    | `boolean`    | No       | -               | -                 |

## Examples

### Basic Usage

```tsx
import { Flex } from '@ybouhjira/hyperkit';

<Flex>Content</Flex>;
```
