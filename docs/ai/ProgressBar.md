| `value`         | `number`  | No       | -       | -           |
| `indeterminate` | `boolean` | No       | -       | -           |
| `size`          | `'sm'     | 'md'     | 'lg'`   | No          | -   | -   |
| `color`         | `string`  | No       | -       | -           |
| `class`         | `string`  | No       | -       | -           |
| `unstyled`      | `boolean` | No       | -       | -           |
| `aria-label`    | `string`  | No       | -       | -           |
