# Select

> Component

## Props

| Prop          | Type                      | Required | Default | Description |
| ------------- | ------------------------- | -------- | ------- | ----------- |
| `options`     | `SelectOption[]`          | Yes      | -       | -           |
| `value`       | `string`                  | No       | -       | -           |
| `onChange`    | `(value: string) => void` | No       | -       | -           |
| `placeholder` | `string`                  | No       | -       | -           |
| `disabled`    | `boolean`                 | No       | -       | -           |
| `class`       | `string`                  | No       | -       | -           |
| `unstyled`    | `boolean`                 | No       | -       | -           |

## Examples

### Basic Usage

```tsx
import { Select } from '@ybouhjira/hyperkit';

<Select>Content</Select>;
```
